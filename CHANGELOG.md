# Изменения в версии 0.1.0-alpha

## Добавлено

- **Многослойная архитектура**: Проект разделен на слои (Core, Service, Infrastructure, Api) для обеспечения слабой
  связности и соблюдения принципов Чистой Архитектуры;
- **REST API для вопросов**: Реализованы эндпоинты для управления базой вопросов: получение списка, поиск по ID,
  создание и удаление;
- **Механизм генерации билетов**: Разработан сервис `ExamPaperGenerator`, который формирует билеты путем случайной
  выборки вопросов на основе заданных настроек;
- **Экспорт в PDF**: Интегрирована стратегия экспорта на базе QuestPDF с поддержкой нумерации страниц, заголовков и
  оформления билетов;
- **JSON Экспортер**: Добавлена возможность выгрузки структуры билетов в формате JSON для программной обработки;
- **Инфраструктура хранения**: Реализован репозиторий, работающий с локальным JSON-файлом;
- **Автоматизация (Workflows)**: Настроены GitHub Actions для автоматической сборки, тестирования, форматирования кода и
  генерации документации;
- **Комплексное тестирование**: Добавлены модульные тесты для логики генерации и архитектурные тесты для контроля
  зависимостей между проектами.

## Изменено

- **Структура решения**: Проект переведен на использование современного формата решения Visual Studio (.slnx) и разделен
  на отдельные библиотеки классов по слоям.
